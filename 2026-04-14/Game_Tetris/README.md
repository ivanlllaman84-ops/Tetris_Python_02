# Tetris (PyGame)

Небольшая реализация классического Тетриса на `pygame`:
- поле `10x20`;
- 7 стандартных фигур (`I, O, T, L, J, S, Z`);
- поворот, боковое движение, мягкое ускорение падения;
- подсчет очков за очищенные линии;
- рестарт после окончания игры.

## Требования

- Python 3.10+
- `pygame`

Установка зависимостей из корня репозитория:

```bash
pip install -r requirements.txt
```

## Запуск

Из корня репозитория:

```bash
python Game_Tetris/main.py
```

Или из папки `Game_Tetris`:

```bash
python main.py
```

## Управление

- `Left / Right` - перемещение фигуры
- `Up` - поворот фигуры
- `Down` (удерживать) - мягкое ускорение падения
- `R` - перезапуск после `Game Over`

## Правила и механика

- Фигура автоматически падает с базовым интервалом `500 ms`.
- При удержании `Down` интервал падения уменьшается до `50 ms`.
- Когда фигура больше не может опускаться, она фиксируется в поле.
- Полностью заполненные линии удаляются.
- За каждую очищенную линию начисляется `100` очков.
- Игра заканчивается, если новая фигура не может появиться в верхней части поля.

## Параметры окна

- Размер клетки: `30x30` px
- Игровое поле: `10x20` клеток (`300x600` px)
- Правая панель: `180` px (счет и подсказки управления)
- Частота обновления: `60 FPS`
